# 04. UI & 캐릭터 시스템

## UI 구조 개요

```
┌─ 메뉴바 (항상 보임) ──────────┐
│   🐱 67%                       │   ← 가장 활발한 디바이스 + 진행률
└────────────────────────────────┘
            ↓ 클릭
┌─ 팝오버 (메인 화면) ──────────┐
│   디바이스 카드 리스트          │
│   합계 영역                     │
│   설정 / 도움말 진입점          │
└────────────────────────────────┘

별도:
- 데스크톱 위젯 (선택, Phase 1.5)
- 알림 (한도 임박, 업적 해금)
- 설정 윈도우 (계정·캐릭터 관리)
```

## 메뉴바 아이콘

상황별 아이콘 변화:

| 상태 | 표시 | 비고 |
|---|---|---|
| 전부 유휴 | `🐱💤` | 가장 캐릭터 + 자는 표시 |
| 활성 작업중 | `🐱 67%` | 활발한 디바이스 + 5h 진행률 |
| 한도 임박 | `🥵 92%⚠️` | 빨간 경고 색 |
| 오프라인 | `🐱 ⚪` | 흐릿한 아이콘 |
| 계정 여러개 (Phase 2) | `🐱+2` | 메인 + 다른 계정 N개 |

**기술**: `NSStatusItem` + `NSImage` 동적 갱신. SF Symbols로 보조 표시.

### 멀티 계정 (Phase 2)

상단 탭으로 계정 전환:

```
[ 내 가족 ] [ 동료 A ] [ 동료 B ] [ 모두 ]
              ──────
```

또는 한 화면에 색상으로 구분:

```
🐱 (파랑) 맥북프로  ─ 내 계정
🐶 (파랑) 데스크탑  ─ 내 계정
🐰 (분홍) 동료A맥   ─ 동료 A
🐼 (초록) 동료B맥   ─ 동료 B
```

## 캐릭터 시스템

### 캐릭터 풀 (Phase 1 시작 라인업)

기본 16종 (이모지 베이스):
```
🐱 🐶 🦊 🐰 🐼 🦝 🐧 🐸
🦄 🐢 🐙 🦔 🦦 🐻 🐯 🐷
```

추가 가능 (Phase 1.5+): 🦖 🐉 🦋 🐝 🦉 🐺

### 표정/상태 매핑

각 캐릭터는 상태별 뷰를 가짐:

| Mood | 표정 | 말풍선 예시 | 색상 톤 |
|---|---|---|---|
| `working` | 🔥 (활발) | "지금 코딩중!" / "집중!" / "GO!" | 노랑 |
| `happy` | 😊 (기본) | "잘 되고있어!" / "쾌적함!" | 초록 |
| `sleepy` | 💤 (졸림) | "zzz..." / "낮잠..." | 파랑 |
| `confused` | 🤔 (갸웃) | "/clear 어때?" / "음..." | 보라 |
| `stressed` | 😰 (걱정) | "토큰 많이 먹었어..." | 주황 |
| `danger` | 🥵 (위험) | "잠깐만 쉬자!!" | 빨강 |
| `offline` | ⚫ (흐림) | "어디 갔어요...?" | 회색 |

말풍선 텍스트는 mood + 캐릭터별로 여러 후보에서 랜덤 선택 → 매번 살짝 다름 → 살아있는 느낌.

### 상태 결정 로직 (요약)

```swift
enum Mood { case working, happy, sleepy, confused, stressed, danger, offline }

func computeMood(_ device: Device) -> Mood {
    guard device.isOnline else { return .offline }
    if device.fiveHourUsage > 0.90 { return .danger }
    if device.fiveHourUsage > 0.75 { return .stressed }
    if device.currentSessionActive { return .working }
    if device.cacheHitRate < 0.3 && device.currentTokens > 50_000 { return .confused }
    if device.minutesSinceLastActivity > 30 { return .sleepy }
    return .happy
}
```

### 애니메이션 (Phase 1.5+)

- **숨쉬기**: idle 캐릭터가 약간씩 위아래로 (1초 주기)
- **놀라기**: mood가 갑자기 변할 때 0.3초 점프
- **눈 깜빡**: 5~10초마다 (선택)
- **수면**: sleepy일 때 z 표시 페이드 인/아웃

기술: SwiftUI `withAnimation` + `Lottie iOS` (선택). 처음엔 단순 transform만.

## 디바이스 카드 인터랙션

| 액션 | 결과 |
|---|---|
| 카드 hover | 살짝 확대 + 그림자 |
| 카드 클릭 | 상세 뷰 (오늘 사용 그래프, 세션 리스트) |
| 카드 우클릭 | 컨텍스트 메뉴 (이름 변경, 캐릭터 변경, 삭제) |
| 캐릭터 더블클릭 | 작은 애니메이션 (꼬리 흔들기 등) |
| 말풍선 클릭 | 다음 멘트로 전환 |

## 알림 (macOS Notification Center)

| 트리거 | 알림 메시지 |
|---|---|
| 5h 80% 도달 | "🥵 맥북프로가 80% 썼어요. 잠깐 쉬어가는 거 어때요?" |
| 5h 95% | "🚨 한도 임박! 저장하고 잠시 멈춰주세요." |
| 캐시 효율 < 20% & 토큰 > 100K | "🤔 맥북프로: /clear 명령어로 새 세션 시작 추천" |
| 업적 해금 | "🏆 캐시 마스터 달성!" |

알림은 사용자 설정에서 종류별 토글.

## 디자인 가이드라인

- **모서리 반경**: 12pt (큰 카드), 8pt (작은 요소)
- **간격**: macOS HIG 8pt 그리드
- **타이포그래피**: SF Pro (system), 강조에 SF Pro Rounded
- **색상**: macOS Dynamic Colors (다크/라이트 자동)
- **이모지**: Apple Color Emoji 그대로
- **여백**: 답답하지 않게, 메뉴바 팝오버 너비 380pt

## 접근성

- VoiceOver로 캐릭터 상태 읽기 ("맥북프로, 활성, 67% 사용중")
- Dynamic Type 지원
- 색상에만 의존 X (텍스트도 함께)

## 미정 디자인 결정

- [ ] 캐릭터 에셋 — 이모지 그대로 vs 커스텀 일러스트 vs Lottie
- [ ] 말풍선 멘트 톤 — 귀여움 / 차분함 / 유머 비율
- [ ] 알림 빈도 — 너무 잦으면 끔, 적으면 효용 떨어짐
- [ ] 다크모드에서 캐릭터 가독성 (이모지는 자동, 일러스트는 별도)
