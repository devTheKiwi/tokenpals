# 🐾 TokenPals

> 다마고치 + Claude Code 토큰 모니터.
> 디바이스마다 귀여운 캐릭터를 붙이고, 토큰 사용량을 실시간으로 한 곳에서 본다.

## 한 줄 소개

Claude Code Max 구독 사용자를 위한 **macOS 메뉴바 앱**. 여러 디바이스와 (나중에) 여러 계정의 토큰 사용량을 한 화면에서 모니터링하고, 디바이스별로 캐릭터·이름을 붙여 상태를 귀엽게 표현한다.

## 왜 만드나

- Claude Code Max는 5h 윈도우·주간 한도가 있는데, **여러 디바이스에서 쓰면 합산이 안 보임**
- 기존 OSS 도구는 단일 디바이스만 지원하거나 UI가 너무 기술적
- 토큰 절약하려면 "어디서 얼마나 쓰고 있는지"가 한눈에 보여야 함
- 결정적으로 — **귀여워야 매일 봄**

## 핵심 컨셉

각 디바이스 = 캐릭터 한 마리.
캐릭터는 그 디바이스의 상태(활성/유휴/위험/오프라인)에 따라 **표정과 말풍선이 변함**.

```
🐱 맥북프로   "지금 코딩중!"   ████████░░ 67%
🐶 데스크탑   "낮잠 자는중..."  ██░░░░░░░░ 13%
🦊 노트북     "어디 갔어요?"    오프라인
```

## 기술 스택 (요약)

- **앱**: SwiftUI 메뉴바 앱 (macOS 14+)
- **백엔드**: Supabase (Postgres + Auth + Realtime, 매니지드)
- **데이터 소스**: `~/.claude*/projects/*.jsonl` 파일 워처
- **배포**: 우선 ad-hoc 서명 .dmg → 나중에 Apple Developer
